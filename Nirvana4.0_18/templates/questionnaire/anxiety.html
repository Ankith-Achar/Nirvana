Anxiety Assessment

This questionnaire is designed to help assess symptoms commonly associated with anxiety. Please answer each question honestly, considering how you've felt over the past two weeks.

Important: This is not a diagnostic tool. If you're experiencing distress, please consult a mental health professional.

2. Not being able to stop or control worrying

3. Worrying too much about different things

4. Trouble relaxing

5. Being so restless that it is hard to sit still

7. Feeling afraid, as if something awful might happen
